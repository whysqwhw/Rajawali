@NonNull fun setToOrthographic2D(x: Double, y: Double, width: Double, height: Double): Matrix4

Sets this Matrix4 to an orthographic projection matrix with the origin at (x,y) extended to the specified width and height. The near plane is at 0 and the far plane is at 1.

Parameters

x - double The x coordinate of the origin.

y - double The y coordinate of the origin.

width - double The width.

height - double The height.

Return
A reference to this Matrix4 to facilitate chaining.

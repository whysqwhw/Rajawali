fun getGimbalPole(): Int
fun getGimbalPole(): Int

Get the pole of the gimbal lock, if any. Requires that this Quaternion be normalized.

Return
positive (+1) for north pole, negative (-1) for south pole, zero (0) when no gimbal lock

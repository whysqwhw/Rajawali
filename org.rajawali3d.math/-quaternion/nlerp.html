nlerp

@NonNull static fun nlerp(@NonNull q1: Quaternion, @NonNull q2: Quaternion, t: Double, shortestPath: Boolean): Quaternion
@NonNull static fun nlerp(@NonNull q1: Quaternion, @NonNull q2: Quaternion, t: Double, shortestPath: Boolean): Quaternion

Performs normalized linear interpolation between two Quaternions and creates a new one for the result.

Parameters

q1 - Quaternion The starting point.

q2 - Quaternion The destination point.

t - double The interpolation value. [0-1] Where 0 represents q1 and 1 represents q2.

shortestPath - boolean indicating if the shortest path should be used.

Return
Quaternion The normalized interpolated Quaternion.

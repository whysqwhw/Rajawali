open fun replaceScene(scene: Scene, location: Int): Boolean

Replaces a Scene in the renderer at the specified location in the list. This does not validate the index, so if it is not contained in the list already, an exception will be thrown. If the Scene being replaced is the one in current use, the replacement will be selected on the next frame.

Parameters

scene - Scene object to add.

location - Integer index of the Scene to replace.

Return
boolean True if the replace task was successfully queued.

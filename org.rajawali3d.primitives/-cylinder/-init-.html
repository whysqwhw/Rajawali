Creates a cylinder primitive. Calling this constructor will create texture coordinates but no vertex color buffer.

Parameters

length - The length of the cylinder

radius - The radius of the cylinder

segmentsL - The number of length segments

segmentsC - The number of circle segments

Cylinder(length: Float, radius: Float, segmentsL: Int, segmentsC: Int, createTextureCoordinates: Boolean, createVertexColorBuffer: Boolean, createVBOs: Boolean)

Creates a cylinder primitive.

Parameters

length - The length of the cylinder

radius - The radius of the cylinder

segmentsL - The number of length segments

segmentsC - The number of circle segments

createTextureCoordinates - A boolean that indicates whether the texture coordinates should be calculated or not.

createVertexColorBuffer - A boolean that indicates whether a vertex color buffer should be created or not.

createVBOs - A boolean that indicates whether the VBOs should be created immediately.

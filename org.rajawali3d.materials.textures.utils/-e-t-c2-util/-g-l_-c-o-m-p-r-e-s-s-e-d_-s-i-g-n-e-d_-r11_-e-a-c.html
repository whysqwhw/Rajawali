GL_COMPRESSED_SIGNED_R11_EAC

static val GL_COMPRESSED_SIGNED_R11_EAC: Int
static val GL_COMPRESSED_SIGNED_R11_EAC: Int

A one-channel signed format. This is good in situations when it is important to be able to preserve zero exactly, and still use both positive and negative values. It is designed to be similar enough to #GL_COMPRESSED_R11_EAC so that hardware can decode both with minimal overhead, but it is not exactly the same. For details, see the corresponding sections of the OpenGL ES 3.0 spec.

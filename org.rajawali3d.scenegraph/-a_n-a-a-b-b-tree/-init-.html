Constructor to setup root node with specified merge/split and grow/shrink behavior.

Parameters

maxMembers - int containing the divide threshold count. When more members than this are added, a partition will divide into 8 children.

minMembers - int containing the merge threshold count. When fewer members than this exist, a partition will recursively merge to its ancestors.

overlap - int containing the percentage overlap between two adjacent partitions. This allows objects to be nested deeper in the tree when they would ordinarily span a boundary.

@NonNull open fun lerp(@NonNull target: Vector3, t: Double): Vector3

Performs a linear interpolation between this Vector3 and to by the specified amount. The result will be stored in the current object which means that the current x, y, z values will be overridden.

Parameters

target - Vector3 Ending point.

t - double [0-1] interpolation value.

Return
A reference to this Vector3 to facilitate chaining.

abstract fun createRenderer(): ISurfaceRenderer
abstract fun createRenderer(): ISurfaceRenderer

Creates the ISurfaceRenderer to use in this display. Optionally null can be returned by displays that do not intend to display a rendered scene. Returning null will cause a warning to be logged to the console in the event null is in error.

Return
ISurfaceRenderer The renderer which will be assigned to the GLSurfaceView in this display, or null.

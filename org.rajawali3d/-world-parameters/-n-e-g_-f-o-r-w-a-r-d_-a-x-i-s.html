NEG_FORWARD_AXIS

static val NEG_FORWARD_AXIS: Vector3
static val NEG_FORWARD_AXIS: Vector3

Global Negative Forward Axis. Defaults to OpenGL -Z axis. It is not safe to modify this Vector3 directly. Instead, use #setWorldAxes(Vector3, Vector3, Vector3).

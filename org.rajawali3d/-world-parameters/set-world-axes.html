setWorldAxes

static fun setWorldAxes(right: Vector3, up: Vector3, forward: Vector3): Unit
static fun setWorldAxes(right: Vector3, up: Vector3, forward: Vector3): Unit

Sets the world axis values after checking that they are all orthogonal to each other. The check performed is to verify that the cross product between right and up is equivilant to forward withing 1ppm error on each component.

Parameters

right - Vector3 The desired right vector. Must be normalized.

up - Vector3 The desired up vector. Must be normalized.

forward - Vector3 The desired forward vector. Must be normalized.

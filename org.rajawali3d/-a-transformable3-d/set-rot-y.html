open fun setRotY(rotY: Double): ATransformable3D
open fun setRotY(rotY: Double): ATransformable3D

Adjusts the rotation of this ATransformable3D by the rotation described by the provided Euler angle. If this is part of a scene graph, the graph will be notified of the change.

Parameters

rotY - double The yaw angle in degrees.

Return
A reference to this ATransformable3D to facilitate chaining.

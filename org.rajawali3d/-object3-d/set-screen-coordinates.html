open fun setScreenCoordinates(x: Double, y: Double, viewportWidth: Int, viewportHeight: Int, eyeZ: Double): Unit

Positions the object in the plane of the screen/viewport (z = 0), with x and y specified in GL screen coordinates (origin at bottom left); Assumes: - the object is a direct child of the current scene (not in a container) - the current viewport is centered on the Z-axis, and has the given dimensions - the current camera is positioned at the given eyeZ

Parameters

x - object x coordinate in pixels [0 = viewport left]

y - object y coordinate in pixels [0 = viewport bottom]

viewportWidth - width of current viewport in pixels

viewportHeight - height of current viewport in pixels

eyeZ - location of current camera on Z-axis

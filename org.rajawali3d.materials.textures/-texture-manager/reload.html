fun reload(): Unit
fun reload(): Unit

Restores all textures that are managed by the TextureManager. All textures will be restored when the OpenGL context is being recreated. This however needs to be indicated by setting ATexture#shouldRecycle(boolean) to true (which is the default). It will then keep a reference to the Bitmap which means the application will take up more memory.

The advantage of storing the Bitmap in memory is that it the texture can quickly be recovered when the context is restored.

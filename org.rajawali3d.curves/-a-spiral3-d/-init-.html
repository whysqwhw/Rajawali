ASpiral3D(density: Double, start: Vector3, normal: Vector3, spiralIn: Boolean)
ASpiral3D(density: Double, start: Vector3, normal: Vector3, spiralIn: Boolean)

Constructs a ArchimedeanSpiral3D with the specified parameters.

Parameters

density - double Factor which determines how tightly the spiral is curled.

start - Vector3 The point where the spiral should start from.

normal - Vector3 The normal vector of the plane the spiral is in. This is assumed to be orthogonal to the vector formed from the start to the origin.

spiralIn - boolean True if the spiral should move from the staring point in. False to move from starting point out.

ICurve3D spiral implementation following an Archimedean spiral. Special values of the density parameter result in special conditions:

|-------------|---------| | Spiral Name | density | |-------------|---------| | lituus | -2 | | hyperbolic | -1 | | Archimedes' | 1 | | Fermat's | 2 | |-------------|---------|

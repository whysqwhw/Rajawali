reparametrizeForUniformDistribution

open fun reparametrizeForUniformDistribution(resolution: Int): Unit
open fun reparametrizeForUniformDistribution(resolution: Int): Unit

Creates a curve with uniformly distributed points. Please note that this might alter the shape of the curve slightly. The higher the resolution, the more closely it will resemble to original curve. You'd typically want to use this for smooth animations with constant speeds. 

 myCurve.reparametrizeForUniformDistribution(myCurve.getPoints().size() * 4);

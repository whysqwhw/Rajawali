open fun calculatePoint(result: Vector3, theta: Double): Unit

Overrides ICurve3D.calculatePoint

Calculates the position on the spiral for the specified polar angle. This takes an additional parameter of a Vector3 which will be set to the calculated position.

Parameters

result - Vector3 to set with the updated position.

theta - double the polar angle to calculate for, in degrees.

open fun setRepeatCount(repeatCount: Int): Unit

Set the number of times to repeat the animation. Repeat count will be ignored for RepeatMode#NONE, RepeatMode#INFINITE, and RepeatMode#REVERSE_INFINITE. When using RepeatMode#REVERSE each direction change will be counted against the repeat count.

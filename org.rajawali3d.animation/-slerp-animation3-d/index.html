SlerpAnimation3D

open class SlerpAnimation3D : Animation3D
open class SlerpAnimation3D : Animation3D

Performs spherical linear interpolation (SLERP) animation between two Vector3s. Example usage: 

 SlerpAnimation3D anim = new SlerpAnimation3D(pointOnSphere1, pointOnSphere1); anim.setDuration(1000); anim.setInterpolator(new AccelerateDecelerateInterpolator()); anim.setTransformable3D(myObject); registerAnimation(anim); anim.play();

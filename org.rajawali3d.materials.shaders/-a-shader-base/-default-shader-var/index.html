DefaultShaderVar

class DefaultShaderVar : IGlobalShaderVar
class DefaultShaderVar : IGlobalShaderVar

The default shader variables are used in the default vertex and fragment shader. They define variables for matrices, position, texture attributes, etc. These shader variables can be used by custom shaders as well. When one of these variables is required the AShader#getGlobal(IGlobalShaderVar) method can be called. For instance: 

 // (in a class that inherits from AShader): RVec4 position = (RVec4) getGlobal(DefaultShaderVar.G_POSITION);
